The slip ring transfers data across a rotating joint.

Backing the strips with a rigid material helps protect the LEDs from broken connections.

{% include youtubePlayer.html id="_z-agriuTms" %}

Leaving one end of the rotation axis unpinned can result in harmonic vibration.

{% include youtubePlayer.html id="EMkDjR-ZdeM" %}

The LEDs need a fast refresh rate to create the smallest arcs possible.  Changing the speed of the rotation affects the resolution of the strip.

{% include youtubePlayer.html id="2fc0eFDiKQo" %}

Distributing the LED bars around the axis can mitigate the vibration, but it may not eliminate the problem.

{% include youtubePlayer.html id="n6S9gD-Kurw" %}
